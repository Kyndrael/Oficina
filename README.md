# 🛠️ Sistema de Gerenciamento de Ordens de Serviço - Oficina Mecânica

## 📌 Descrição

Este projeto representa um **esquema conceitual de banco de dados** para um sistema de controle e execução de ordens de serviço em uma oficina mecânica. O sistema foi projetado com base em uma narrativa proposta, visando gerenciar veículos, ordens de serviço, peças, serviços realizados e equipes de mecânicos.

## 🧠 Contexto da Narrativa

- Clientes levam seus veículos à oficina para revisões ou consertos.
- Cada veículo é atribuído a uma equipe de mecânicos que gera uma OS.
- A OS contém data de emissão, conclusão, status, valores e os serviços e peças utilizados.
- A mesma equipe realiza e avalia os serviços.
- Valores são calculados a partir de uma tabela de mão-de-obra e peças.
  

## 📊 Entidades Principais

- Cliente
- Veículo
- Ordem de Serviço (OS)
- Equipe
- Mecânico
- Serviço
- Peça
